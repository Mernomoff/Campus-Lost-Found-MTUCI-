# Campus: Потеряно/Найдено (МТУСИ)

Веб-приложение для студентов МТУСИ: публикуй объявления о потерянных/найденных вещах, ищи, чатайся и смотри на карте.

## Стек
- Backend: Flask (Python)
- Frontend: Jinja шаблоны (планирую React)
- Карта: Leaflet.js
- Будущее: Мобильное app (React Native?)

## Установка
1. Клонируй: `git clone https://github.com/твой-username/Campus-Lost-Found-MTUCi.git`
2. Установи: `pip install flask`
3. Запусти: `python app.py`
4. Открой: http://localhost:5000

## Фичи
- Главная: Список объявлений
- Добавить: Форма (тип, категория, описание, локация)
- Поиск: По описанию/категории
- Детали: + Чат
- Карта: Маркеры и полигон кампуса
- Профиль: Приватность
